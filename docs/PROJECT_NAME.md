# 项目名称说明

## 🏷️ 项目名称

### 中文名称
**宿舍大师**

- **含义**：
  - "宿舍"：明确项目定位，专注于宿舍管理
  - "大师"：体现专业性和强大功能，寓意管理得心应手

- **优势**：
  - ✅ 简洁易记
  - ✅ 朗朗上口
  - ✅ 体现功能定位
  - ✅ 符合中文命名习惯

### 英文名称
**DormMaster**

- **构成**：
  - `Dorm`：Dormitory（宿舍）的缩写
  - `Master`：大师、专家

- **优势**：
  - ✅ 国际化，易于国际推广
  - ✅ 简洁专业
  - ✅ 易于搜索和识别
  - ✅ 符合开源项目命名惯例

### 缩写
**DM**

- 用于代码注释、版本号等简短场景

## 🎯 命名理由

### 为什么选择"宿舍大师"？

1. **功能定位清晰**
   - 直接表明是宿舍管理系统
   - 用户一眼就能理解项目用途

2. **品牌感强**
   - "大师"体现专业性和权威性
   - 给用户可靠、强大的印象

3. **易于传播**
   - 简洁易记，便于口头传播
   - 适合各种场景（文档、代码、演示）

4. **差异化**
   - 与简单的"宿舍管理系统"区分
   - 突出产品的高级特性

### 为什么选择"DormMaster"？

1. **国际化**
   - 英文名便于国际推广
   - 符合开源项目惯例

2. **技术友好**
   - 适合作为 Git 仓库名
   - 适合作为域名（dormmaster.io）
   - 适合作为产品品牌

3. **专业性**
   - 简洁专业，不幼稚
   - 适合企业级应用
