🚀 快速部署（3步完成）
================================================================================

第1步：上传文件
    → 将所有文件上传到网站根目录
    → 例如：/www/wwwroot/10.10.70.17_8000/
    → 确保目录结构完整

第2步：运行修复工具
    → 浏览器访问：http://10.10.70.17:8000/一键修复完整版.php
    → 填写数据库信息（主机、端口、数据库名、用户名、密码）
    → 点击"开始修复"
    → 等待自动完成（约30秒）

第3步：登录使用
    → 访问：http://10.10.70.17:8000/
    → 登录：admin / admin123
    → 立即修改密码！

🔧 如果遇到问题
================================================================================

1. 无法访问修复工具？
   → 检查文件是否上传完整
   → 尝试访问：http://10.10.70.17:8000/install_fixed.php

2. 数据库连接失败？
   → 确保数据库已创建
   → 检查用户名密码是否正确
   → 检查是否有远程访问权限

3. 登录失败？
   → 运行修复工具自动修复
   → 检查config/config.php是否存在
   → 查看logs/error_*.log

4. 页面显示错误？
   → 检查PHP版本（需要7.4+）
   → 查看错误日志
